# MatchLog

축구 직관 기록 및 일정 관리 앱

## 주요 기능

- **직관 일기**: 경기 직관 기록 작성 (사진, 좌석, 평점, MVP 등)
- **경기 일정**: 리그별 경기 일정/라인업/통계/타임라인
- **국가대표**: 대한민국 국가대표 일정, 2026 월드컵 카운트다운
- **커뮤니티**: 직관 후기 공유, 경기별 게시글 필터링
- **즐겨찾기**: 팀/선수 등록 및 일정 알림
- **순위표**: 리그별 순위 및 팀 통계

## 기술 스택

- Flutter + Firebase (Auth, Firestore, Storage)
- Riverpod (상태 관리)
- GoRouter (라우팅)
- TheSportsDB API (경기 데이터)

## 지원 리그
